# Product Brief

## Working Name

Token Arcade

## One-Liner

Turn AI coding token usage into a cozy pixel arcade full of coins, cabinets, badges, and prizes.

## Problem

AI coding tools produce a lot of local history and token usage data, but most tools show that data as logs, charts, or cost dashboards. That is useful, but not emotionally rewarding.

Power users often spend many tokens across many projects. Token Arcade makes that usage feel tangible and playful without asking the user to inspect every conversation.

## Product Fantasy

The user has a private arcade powered by their AI usage.

Each project is an arcade cabinet. When a project consumes tokens, the cabinet gains lights, levels, and style. The user's total token usage becomes coins they can spend on collectibles and room upgrades.

The app should feel like opening a small game after a coding session and collecting the rewards from the work they already did.

## Target User

AI coding power users who use Claude Code, Codex, Gemini CLI, or similar tools across multiple local projects.

They enjoy:

- visible progress
- collectibles
- cozy pixel art
- lightweight rituals
- seeing their AI usage history without reading transcripts

## Core Product Bet

Token usage alone is enough to create a satisfying game loop if the conversion and reward moments are delightful.

This product should not try to prove the user's work was productive. It should only celebrate that energy was spent.

## Differentiation

Many existing projects are token dashboards, history viewers, XP overlays, or RPG-style token games. Token Arcade should be different:

- more visual than a dashboard
- lighter than an RPG
- more collectible than a usage monitor
- more playful than a history viewer
- focused on project cabinets, prizes, and arcade-room growth

## North Star

The user opens the app after using AI coding tools and thinks:

"Nice. My project cabinet leveled up, I got coins, and I can unlock something."
